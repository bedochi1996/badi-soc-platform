# BADI SOC — Saudi-Native SOC Operations Simulation Platform

BADI SOC is a proof-of-concept SOC operations simulation platform designed for Arabic-speaking cybersecurity analysts and Saudi cybersecurity environments.

The platform focuses on practical SOC workflows including alert triage, incident investigation, MITRE ATT&CK mapping, playbooks, reports, threat mapping, SLA tracking, analyst scoring, and NCA ECC-oriented compliance views.

> **Status:** Portfolio Project / Proof of Concept
> **Purpose:** Demonstrate SOC workflow understanding, Blue Team operations, and Saudi-focused cybersecurity platform design.

---

## Key Capabilities

- Alert Triage dashboard with severity classification
- Incident investigation workspace
- War Room collaboration simulation
- MITRE ATT&CK mapping view
- Playbooks for common incident types
- Threat map and live attack feed
- SLA tracking by alert severity
- Analyst scoreboard and training mode
- Attack replay scenarios
- NCA ECC-oriented compliance mapping
- Arabic-first UI for SOC analysts

---

## SOC Workflows Covered

- Alert intake
- Severity classification
- False positive vs true positive analysis
- Incident escalation
- Timeline reconstruction
- IOC review
- MITRE ATT&CK technique mapping
- Playbook-driven response
- Report generation
- Analyst performance tracking
