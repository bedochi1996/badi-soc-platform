# Templates

This directory contains templates for SOC operations and documentation.

## Available Templates

### SIEM Query Templates
- **Suspicious Login Detection**: Detects failed login attempts and unusual access patterns
- **Malware Activity**: Identifies potential malware execution and C2 communication
- **Data Exfiltration**: Monitors for large data transfers and suspicious file access
- **Lateral Movement**: Tracks unusual network connections and privilege escalation

### Incident Response Templates
- **Initial Assessment**: Quick triage checklist for new incidents
- **Evidence Collection**: Standardized procedures for gathering forensic data
- **Containment Actions**: Step-by-step containment procedures
- **Post-Incident Review**: Template for lessons learned documentation

### Alert Playbooks
- **Phishing Investigation**: Email-based threat investigation procedures
- **Ransomware Response**: Immediate response to ransomware detection
- **Insider Threat**: Investigation procedures for internal threats
- **DDoS Attack**: Distributed denial of service mitigation steps

### Compliance Templates
- **NCA ECC Reporting**: Saudi NCA Essential Cybersecurity Controls reporting format
- **Incident Notification**: External notification templates
- **Risk Assessment**: Security risk documentation templates

## Usage

These templates are designed to be customized for your specific environment and requirements. Copy and modify as needed for your SOC operations.
